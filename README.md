Description

Projet de formation permettant à l’utilisateur de personnaliser les couleurs de la page (fond et texte) et de conserver ses choix grâce aux cookies.

Fonctionnalités

Choix du fond et de la couleur du texte via un formulaire.

Les choix sont enregistrés dans des cookies pour conserver le thème au rechargement de la page.

Possibilité de revenir au thème initial ou de réinitialiser les choix.

Les cookies sont valides pendant 1 an.

Technologies

PHP

HTML

CSS minimal

Installation

Cloner le dépôt :

git clone https://github.com/ton-username/theme-selector-php.git


Placer le projet dans votre serveur local (ex : MAMP, XAMPP).

Accéder au fichier principal dans le navigateur.

Utilisation

Sélectionner les couleurs souhaitées pour le fond et le texte.

Cliquer sur “Apply” pour enregistrer le thème.

Cliquer sur “Reset” ou “Come back to initial colors” pour revenir aux couleurs par défaut.

Objectif pédagogique

Apprendre à manipuler les cookies en PHP.

Comprendre la persistance des choix utilisateur entre les rechargements de page.

Pratiquer l’intégration PHP + HTML + CSS pour créer une interface interactive.
